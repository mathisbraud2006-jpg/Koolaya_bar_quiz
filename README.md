# 🍹 Koolaya Bar Quiz

A fun and interactive bar quiz game built with HTML. Test your knowledge and challenge your friends with trivia questions!

## 📋 Description

Koolaya Bar Quiz is a web-based quiz application designed to entertain and challenge players at a bar or any social gathering. It features a collection of trivia questions across various categories, perfect for team competitions or solo play.

## ✨ Features

- 🎯 **Interactive Quiz Gameplay** - Engaging multiple-choice questions
- 🏆 **Score Tracking** - Keep track of your performance
- 🎨 **Responsive Design** - Works on desktop and mobile devices
- 🍺 **Social Fun** - Perfect for group play and friendly competitions
- ⚡ **Instant Feedback** - Immediate answer verification

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required!

### Installation

1. Clone the repository:
```bash
git clone https://github.com/mathisbraud2006-jpg/Koolaya_bar_quiz.git
cd Koolaya_bar_quiz
```

2. Open `index.html` in your browser:
```bash
open index.html
```
Or simply double-click the `index.html` file in your file explorer.

## 📖 How to Play

1. Open the quiz in your browser
2. Read each question carefully
3. Select your answer from the available options
4. Click submit to check your answer
5. View your score at the end of the quiz
6. Challenge your friends to beat your score!
